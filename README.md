# Wello Go Common

团队共用组件工具箱，提供 Kafka、Redis、Logging 等常用组件的封装。

## 目录结构

```
.
├── kafka/       # Kafka 相关组件
├── redis/       # Redis 相关组件
├── logging/     # Logging 相关组件
└── examples/    # 使用示例
```

## 使用方式

```go
import "github.com/wello-go/common/kafka"
import "github.com/wello-go/common/redis"
import "github.com/wello-go/common/logging"
```
